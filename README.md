# todo-react-app



## 📝 프로젝트 개요
React + Zustand + Tailwind CSS 4 + Vite 기반의 Todo 웹 애플리케이션입니다.


## 🛠 개발 환경
- Node.js v20 LTS
- Vite 6
- React 19
- Zustand
- Tailwind CSS 4

## ✅ 구현 기능
### [ 기본 기능 ]
- Todo 추가
- 목록 표시
- 완료 토글
- 삭제 (CRUD)

### [ 확장 기능 ]
- 🌤 날씨 API 연동 (OpenWeatherMap) — 우측 상단 WeatherWidget, .env API 키 관리, 로딩/에러 상태 UI
